## Что делает приложение

Помогает сотруднику собрать корректное письмо в IT-отдел: выбрать тип проблемы, заполнить только нужные поля, получить готовую тему и текст письма. Параллельно подбирает подходящие инструкции из базы Wiki — часто вопрос удаётся решить без письма.

---

## Для кого

- **Обычные пользователи** — заполняют заявку и получают готовое письмо.
- **Сотрудники IT** — используют вкладку «🔧 IT: обновление Wiki» для обновления базы инструкций.

---

## Как запустить

Открыть файл `Заявка_в_IT_отдел_v0.27_20260601_1500.html` двойным кликом в браузере.  
Интернет и установка не требуются. Всё работает локально.

---

## Как пользоваться

### Пользователю

1. Выбрать **город**, **юрлицо**, **программу/базу**, **контакт**.
2. В списке «Ваша проблема» выбрать тип (список отсортирован по алфавиту).
3. Заполнить поля проблемы и описание (что случилось).
4. Справа появятся **«Подходящие инструкции»** — сначала проверьте их, возможно вопрос решается без письма.
5. Нажать **«Сформировать»**, скопировать тему и текст письма, прикрепить файлы вручную.

### IT-сотруднику (вкладка «🔧 IT: обновление Wiki»)

Открывается по паролю `010109`.  
Это мягкая защита от случайного захода — пароль виден в коде страницы, секретным не является.

Шаги обновления базы:
1. Скачать XAR из Wiki.
2. Запустить `xar_to_json.py` (дважды, для каждого файла).
3. Переименовать полученные JSON-файлы.
4. Запустить `update_wiki_public_internal.py`.
5. Собрать новый HTML через `build_wiki_search_app_v3`.
6. Открыть собранный HTML и проверить, что списки открываются и кнопки работают.
7. Раздать обновлённый файл пользователям.

> **Важно:** новый HTML собирать ТОЛЬКО скриптом `build_wiki_search_app_v3`. Вручную не редактировать.

Все три скрипта встроены во вкладку с кнопками «Скопировать скрипт» и «Скачать .py».

## Что делать при ошибках

Если что-то пошло не так:

1. На экране появится **красный блок с текстом ошибки**.
2. Нажать **«Скопировать ошибку»** — скопируется полный текст.
3. Нажать **«Скачать ошибку .txt»** — сохранится файл с подробным описанием. Если браузер блокирует скачивание, скопируйте текст ошибки из поля на экране.
4. Отправить скопированный или скачанный текст в IT-отдел.

Ошибку не нужно переписывать вручную — полный технический текст сохранится в файле или буфере обмена.

## Глобальный экран ошибок

В приложении реализован механизм видимого отображения любой ошибки:

- Ошибка появляется крупным жирным красным текстом поверх интерфейса.
- Рядом — кнопки **«Скопировать ошибку»** и **«Скачать ошибку .txt»**.
- Если скачивание недоступно в браузере — в поле на экране отображается полный текст ошибки для ручного копирования.
- Ошибка не исчезает автоматически — можно закрыть вручную после копирования.
