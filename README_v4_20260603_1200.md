## Что делает приложение

«Заявка в IT отдел» — инструмент для сотрудников ГК Миллениум. Приложение помогает правильно оформить и отправить заявку в IT-отдел: выбрать город, организацию, программу, тип проблемы — и получить готовое письмо с корректной темой и текстом.

Дополнительно приложение подбирает подходящие инструкции из корпоративной Wiki (539 записей) прямо по описанию проблемы — без поиска вручную.

Раздел для IT-администраторов содержит встроенные Python-скрипты для обновления Wiki.

## Как запустить

Откройте файл `index.html` в любом современном браузере (Chrome, Edge, Firefox, Safari).

Никаких установок, серверов, интернета и API-ключей не требуется. Всё работает локально.

## Как пользоваться

### Вкладка «1. Заполнить заявку»

1. Выберите **Город** из списка.
2. Выберите **Организацию** (или оставьте пустым, если не применимо).
3. Выберите **Программу**, с которой возникла проблема.
4. Выберите **Вашу проблему** из выпадающего списка (список отсортирован по алфавиту).
5. При необходимости опишите проблему подробнее в текстовом поле.
6. Прикрепите скриншоты или файлы, если нужно.
7. Нажмите кнопку **Сформировать письмо**.

### Вкладка «2. Что получится»

- Проверьте сформированные **тему письма** и **текст**.
- Нажмите **Скопировать тему** и **Скопировать текст**, затем вставьте в свой почтовый клиент.
- Или нажмите **Открыть в почте** — приложение откроет черновик письма автоматически.

### Вкладка «3. Настройка полей»

- Просмотр и ручная настройка списков городов, организаций, программ и проблем.
- Изменения сохраняются в рамках текущей сессии (без перезагрузки страницы).

### Вкладка «Поиск инструкций»

- Введите ключевые слова или выберите тип проблемы.
- Приложение подберёт подходящие инструкции из базы Wiki (539 записей).
- Подбор учитывает как выбранную проблему, так и текстовое описание.
- Закреплённые ссылки для ключевых проблем (например, «Декларация соответствия») всегда идут первыми.

### Вкладка «Инструкция»

- Полная встроенная справка по работе с приложением.

### Вкладка «🔧 IT: обновление Wiki»

- Раздел защищён паролем (для IT-специалистов).
- Содержит три встроенных Python-скрипта для обновления корпоративной Wiki:
  - `xar_to_json.py` — конвертация XAR-архива в JSON.
  - `update_wiki_public_internal.py` — обновление публичных и внутренних страниц Wiki.
  - `build_wiki_search_app_v3_20260601_1131.py` — сборка приложения поиска по Wiki.
- Скрипты можно скопировать или скачать прямо из интерфейса.

## Что делать при ошибках

Если что-то пошло не так:

1. На экране появится **красный блок с описанием ошибки**.
2. Нажмите **«Скопировать ошибку»** — текст ошибки скопируется в буфер.
3. Нажмите **«Скачать ошибку .txt»** — сохранится файл с полным текстом ошибки.
4. Отправьте этот текст или файл в IT-отдел.

Если кнопки копирования или скачивания недоступны в вашем браузере — выделите и скопируйте текст ошибки вручную из красного блока.
