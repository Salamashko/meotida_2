Taxonomy label for list items — in the app it marks each Wiki instruction's access level. Smaller and squarer than `Pill`.

```jsx
<Badge tone="public">Публичная</Badge>
<Badge tone="internal">Внутренняя</Badge>
<Badge tone="candidate">Кандидат</Badge>
```

Tones: `public` (green), `internal` (amber), `candidate` (indigo). For header metadata/status use `Pill`.
