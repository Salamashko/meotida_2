A small rounded-capsule chip for header metadata and inline status — version tags, copyright, the "works locally" indicator.

```jsx
<Pill>v32 · Простой режим</Pill>
<Pill tone="ok" icon="🔒">Работает локально</Pill>
<Pill tone="warn">Офлайн</Pill>
```

Tones: `default` (outlined grey), `ok` (green), `warn` (amber). For taxonomy labels on list items use `Badge` instead.
